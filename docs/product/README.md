# 产品文档索引

这组文档用于把当前的产品想法从聊天结论沉淀为可复用的项目资料。

当前版本的核心定义是：

- 左侧是主画布，承载原始信息、截图、文本、线索与上下文。
- 右侧是待办列表，作为从画布内容中投影出来的行动层。
- 左右两栏之间存在可视化映射关系，支持 hover 高亮、连线与双向定位。

建议阅读顺序：

1. 先读 PRD，明确产品到底是什么。
2. 再读无限画布决策备忘录，明确“画布”和“无限”分别意味着什么。
3. 接着看信息架构与主流程，确认双栏工作区体验是否成立。
4. 再看 MVP 功能清单与开发排期，确认第一阶段做什么。
5. 再看后端预留架构草案，确认现在的本地方案如何为云同步留口。
6. 最后看 Excalidraw 二开方案和双栏工作区草图，进入实现层讨论。
